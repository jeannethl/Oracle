Necesario:

instantclient-basic-linux.x64-19.15.0.0.0dbru.zip
instantclient-sqlplus-linux.x64-19.15.0.0.0dbru.zip
instantclient-tools-linux.x64-19.15.0.0.0dbru-2.zip

Descargar de: https://www.oracle.com/database/technologies/instant-client/linux-x86-64-downloads.html



Para este ejemplo los binarios fueron alojados en /home/ubuntu


1.-Crear el directorio oracle

$ cd /opt
$ sudo mkdir oracle
$ chown <usuario>:<grupo> oracle/



2.-Instalar el paquete libaio1 y validar el paquete glibc que sea superior a la versión 2.14

$ sudo apt install libaio1
$ sudo apt list | grep glibc



3.-Descomprimir el instantclient y el sqlplus

$ cd /opt/oracle
$ unzip -q /home/ubuntu/INSTANT_CLIENT/instantclient-basic-linux.x64-19.15.0.0.0dbru.zip
$ unzip -q /home/ubuntu/INSTANT_CLIENT/instantclient-sqlplus-linux.x64-19.15.0.0.0dbru.zip
$ unzip -q /home/ubuntu/INSTANT_CLIENT/instantclient-tools-linux.x64-19.15.0.0.0dbru-2.zip




4.-Ir al directorio de los archivos descomprimidos y validar los links simbolicos

cd /opt/oracle/instantclient_19_15
ls -ltr 

Deben aparecer:
libclntsh.so -> libclntsh.so.19.1
libocci.so -> libocci.so.19.1





5.-Ir a la carpeta network y crear el tnsnames.ora

Ejemplo:

$ cd /opt/oracle/instantclient_19_15/network/admin
$ vi tnsnames.ora


TRZS =
  (DESCRIPTION =
    (ADDRESS = (PROTOCOL = TCP)(HOST = 192.168.56.120)(PORT = 1521))
    (CONNECT_DATA =
      (SERVER = DEDICATED)
      (SERVICE_NAME = TRZS)
    )
  )





6.-Establecer las variables de entorno en el .profile del usuario a usar el cliente 

$ cd
$ vi .profile
    export PATH="$PATH:/opt/oracle/instantclient_19_15"
    export LD_LIBRARY_PATH="$LD_LIBRARY_PATH:/opt/oracle/instantclient_19_15"
$ source ~/.profile




7.-Ejecutar el sqlplus 

Ejemplos:

$ sqlplus sys@192.168.56.120:1521/TRZS as sysdba
$ sqlplus sys@TRZS as sysdba
